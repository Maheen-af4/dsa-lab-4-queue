Task 01 — Dynamic Queue Implementation
A program that implements a dynamic queue using C++ class templates with the following features:

Implemented using a class template for type flexibility
Dynamic resizing when queue becomes full
Supports all basic queue operations:

Enqueue()
Dequeue()
IsEmpty()
IsFull()
Peek()
Size()




Task 02 — Reverse First K Elements of a Queue
A program that reverses the first K elements of a queue while keeping the remaining elements in the same relative order.
Allowed queue operations:

enqueue(x)
dequeue()
size()
front()
